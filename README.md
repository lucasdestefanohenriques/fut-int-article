# MDPI-future-internet-paper

This repository contains data and scripts for "A Combined of Advanced Feature Selection Methods for Uncovering Out-profiled Energy Consumption Patterns."

The material is being made available at the request of one of the reviewers.

Some codes were applied recursively, as already explained in the full article.

The data was collected in 383 households, and after preprocessing before applying the clustering algorithms, it was reduced to 373.
